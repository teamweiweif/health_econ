# Causal ML Audit

## Decision

Not run.

## Reason

Causal ML is not justified as a main contribution. The paper should be positioned as a conventional mechanism-oriented SIPP audit or, if support is weak, as a no-go/backup-design memo.

Causal ML was conditional on credible conventional variation, adequate event counts, overlap, stable CATE ranking, and out-of-sample policy value beyond simple rules. The current run does not clear that bar automatically.
