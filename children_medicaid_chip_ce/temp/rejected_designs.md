# Rejected Or Downgraded Designs

- Voluntary-adoption DID: rejected as primary because the 2024 policy is a federal mandate layered on prior state variation, not a clean voluntary adoption design.
- Aggregate enrollment volatility as true churn: rejected because public state-month aggregate data do not observe individual exits and returns.
- Renewal outcomes as child-specific causal outcomes: downgraded because CMS renewal data are generally state-level and not child-specific.
- Causal ML: deferred because treatment timing, pretrends, and mechanism coherence must first pass simpler DDD diagnostics.
- ACS as main design: downgraded because only 2024 is currently available as post-policy validation in the staged public tables.
